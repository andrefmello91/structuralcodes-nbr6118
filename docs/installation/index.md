# Installation

`structuralcodes` is compatible with [![PyPI - Python Version](https://img.shields.io/pypi/pyversions/structuralcodes)](https://pypi.org/project/structuralcodes/) and is installed by typing:

:::::{tab-set}
::::{tab-item} Python
:sync: python

:::{code-block} pwsh
python -m pip install structuralcodes
:::
::::

::::{tab-item} Python launcher
:sync: py

:::{code-block} pwsh
py -m pip install structuralcodes
:::
::::
:::::

If you already have `structuralcodes` installed, you can upgrade it by typing:

:::::{tab-set}
::::{tab-item} Python
:sync: python

:::{code-block} pwsh
python -m pip install --upgrade structuralcodes
:::
::::

::::{tab-item} Python launcher
:sync: py

:::{code-block} pwsh
py -m pip install --upgrade structuralcodes
:::
::::
:::::
